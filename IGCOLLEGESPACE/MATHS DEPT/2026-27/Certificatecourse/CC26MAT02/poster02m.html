Syllabus — Theory (12 hrs) + Practical (18 hrs)
Unit 1 — Python Foundations for Mathematics Theory: 3 h | Practical: 2 h | Total: 5 h
Python syntax, data types, control structures and functions
Introduction to NumPy: arrays, array operations, broadcasting
Mathematical functions in NumPy: linspace, arange, trig, exponential
Plotting with Matplotlib: line plots, scatter, subplots, annotations
Writing and organising mathematical programs in Python
Practical
Array manipulations · Plotting mathematical functions · Writing reusable Python functions for iterative algorithms
Unit 2 — Numerical Methods I: Root Finding Theory: 3 h | Practical: 5 h | Total: 8 h
Errors in numerical computation; convergence criteria
Bisection Method — derivation, algorithm, convergence rate
Regula Falsi (False Position) Method — theory and comparison
Newton-Raphson Method — derivation from Taylor series, quadratic convergence
Comparison of methods: speed, stability, conditions for convergence
Practical
Implement all three root-finding algorithms · Plot iteration convergence graphs · Compare methods on test equations
Unit 3 — Numerical Integration Theory: 3 h | Practical: 5 h | Total: 8 h
Concept of numerical quadrature; Newton-Cotes formulae
Trapezoidal Rule — derivation, error analysis
Simpson's 1/3 Rule — derivation, even-interval requirement
Simpson's 3/8 Rule — derivation, applicability conditions
Comparative accuracy and error estimation
Practical
Implement Trapezoidal, Simpson's 1/3 and 3/8 rules · Visualise area under the curve · Error comparison with exact integrals
Unit 4 — LPP: Graphical Method Theory: 3 h | Practical: 6 h | Total: 9 h
Introduction to Linear Programming Problems (LPP)
Formulation of LPP: objective function and constraints
Graphical method — feasible region, corner-point theorem
Maximisation and minimisation problems
Special cases: unbounded solutions, infeasibility, multiple optima
Practical
Define and solve LPP using PuLP · Plot feasible region and constraints · Highlight corner points and optimal solution
