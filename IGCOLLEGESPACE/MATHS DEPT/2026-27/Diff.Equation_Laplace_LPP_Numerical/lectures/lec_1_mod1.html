🎯 1. Learning Outcomes (CO1)
  • Define an Ordinary Differential Equation (ODE)
  • Determine the order and degree of an ODE
  • Distinguish between linear and nonlinear ODEs
  • Identify ODEs in engineering contexts
  • Apply the Principle of Superposition to linear homogeneous ODEs
🔍 2. Introduction – Why Differential Equations?

Differential equations model nearly every physical phenomenon: population growth, radioactive decay, motion of planets, electrical circuits, heat transfer, and more. When the unknown function depends on one independent variable, it's called an Ordinary Differential Equation (ODE).

📌 3. Definition of ODE
An Ordinary Differential Equation (ODE) is an equation containing an unknown function of one independent variable and its ordinary derivatives.
General form: \[ F(x, y, y', y'', \dots, y^{(n)}) = 0 \] Example: \[ \frac{dy}{dx} = 3x^2 \] where \(x\) is independent, \(y\) dependent.
📈 4. Order of an ODE

The order is the highest derivative present in the equation.

OrderGeneral FormExample
First \(y' = f(x,y)\) \(\frac{dy}{dx} + 2y = e^x\)
Second \(y'' = f(x,y,y')\) \(\frac{d^2y}{dx^2} + 5\frac{dy}{dx} + 6y = 0\)
n-th \(y^{(n)} = f(x, y, y', \dots, y^{(n-1)})\) \(y^{(4)} + 3y'' = \sin x\)

✅ Student Check: Order of \(\frac{d^3y}{dx^3} + 4\frac{dy}{dx} = 0\) is 3rd order.

⚙️ 5. Degree of an ODE

The degree is the power of the highest order derivative (provided equation is polynomial in derivatives).

Example 1: \[ \left(\frac{d^2y}{dx^2}\right)^3 + 5\frac{dy}{dx} = 0 \] → Highest derivative \(y''\) power = 3 → Degree = 3

Example 2: \[ \frac{dy}{dx} = \sqrt{1+x^2} \] → rewrite: \[ \left(\frac{dy}{dx}\right)^2 = 1+x^2 \] → Degree = 2
🔄 6. Linear vs Nonlinear ODEs
Linear ODE: \(y\) and all its derivatives appear only to the first power, no products like \(y \cdot y'\), and coefficients depend only on \(x\). Standard first-order linear form: \[ \frac{dy}{dx} + P(x)y = Q(x) \]
FeatureLinear ODENonlinear ODE
Power of \(y\) & derivativesAlways 1Can be >1 or fractional
Products like \(y\cdot y'\)Not allowedAllowed
Functions like \(\sin y, e^y\)Not allowedAllowed
Superposition principle✓ Applies✗ Does not apply
✔️ Linear examples: \(y' + 2y = x^2\), \(x^2 y'' + x y' -4y=0\)
❌ Nonlinear examples: \(y' = y^2\), \(y \cdot y' = x\), \(\sin y = y''\).
🔀 7. Principle of Superposition
Principle of Superposition (for Linear ODEs):

If \(y_1\) and \(y_2\) are two solutions of a homogeneous linear ODE, then any linear combination \[ y = c_1 y_1 + c_2 y_2 \] is also a solution, where \(c_1\) and \(c_2\) are arbitrary constants.

More generally, if \(y_1, y_2, \dots, y_n\) are solutions of an \(n\)-th order homogeneous linear ODE, then: \[ y = c_1 y_1 + c_2 y_2 + \cdots + c_n y_n \] is also a solution.
Example: Consider the second-order linear homogeneous ODE: \[ y'' - y = 0 \] Check that \(y_1 = e^x\) and \(y_2 = e^{-x}\) are both solutions:

For \(y_1 = e^x\): \(y_1'' = e^x\), so \(y_1'' - y_1 = e^x - e^x = 0\) ✔
For \(y_2 = e^{-x}\): \(y_2'' = e^{-x}\), so \(y_2'' - y_2 = e^{-x} - e^{-x} = 0\) ✔

By the Principle of Superposition, the general solution is: \[ y = c_1 e^x + c_2 e^{-x} \]
⚠️ Important Note: The Principle of Superposition applies only to linear homogeneous ODEs. It does not hold for nonlinear ODEs.

For a non-homogeneous linear ODE \(Ly = g(x)\), if \(y_p\) is a particular solution and \(y_h\) is the general solution of the corresponding homogeneous equation, then: \[ y = y_h + y_p \] is the complete general solution.
📝 8. Solved Examples (Section 1.1 Style)
Example A: \[ \frac{d^2y}{dx^2} + 5\left(\frac{dy}{dx}\right)^3 - 4y = e^x \] ➤ Order = 2, Degree = 1, Nonlinear because \((y')^3\).
Example B: \[ x^2 \frac{d^2y}{dx^2} + 3x \frac{dy}{dx} + 5y = \ln x \] ➤ Order = 2, Degree = 1, Linear.
Example C: \[ \frac{dy}{dx} = \frac{1-y}{1+x} \] → rewrite \((1+x)y' + y = 1\) → Order 1, Degree 1, Linear.
📚 Workout Examples

Problems on order, degree, linearity, and basic ODE concepts. Click Answer to see step-by-step solution.

1. Define an ordinary differential equation. CO1
✅ Solution: An Ordinary Differential Equation (ODE) is an equation that involves an unknown function of a single independent variable and its ordinary derivatives. Formally, \[ F(x, y, y', y'', \dots, y^{(n)}) = 0. \] Example: \(\frac{dy}{dx} = 3x^2\).
2. Find the order and degree of the differential equation \(\frac{d^2y}{dx^2} + 9y = e^{-2x}\). CO1
✅ Step-by-step: The given equation is \(\frac{d^2y}{dx^2} + 9y = e^{-2x}\).
- Highest derivative is \(\frac{d^2y}{dx^2}\) → Order = 2.
- The derivative \(\frac{d^2y}{dx^2}\) appears with power 1 → Degree = 1.
- It is linear because \(y\) and derivatives have power 1, no product terms.
3. Find the order and degree of the differential equation \(\frac{d^2y}{dx^2} + 2\left(\frac{dy}{dx}\right)^{\frac{1}{3}} = 0\). CO1
✅ Step-by-step:

The given equation is: \[ \frac{d^2y}{dx^2} + 2\left(\frac{dy}{dx}\right)^{\frac{1}{3}} = 0 \] Order: The highest derivative present is \(\frac{d^2y}{dx^2}\) → Order = 2.

Degree: The term \(\left(\frac{dy}{dx}\right)^{\frac{1}{3}}\) has a fractional power, so the equation is not polynomial in derivatives as it stands. We must eliminate the fractional power first.

Rearrange: \[ \frac{d^2y}{dx^2} = -2\left(\frac{dy}{dx}\right)^{\frac{1}{3}} \] Cube both sides: \[ \left(\frac{d^2y}{dx^2}\right)^3 = -8\,\frac{dy}{dx} \] Now the equation is polynomial in derivatives. The highest order derivative is \(\frac{d^2y}{dx^2}\) and its power is 3.

∴ Order = 2, Degree = 3.
⚠️ Common Mistakes to Avoid
MistakeCorrection
"Degree = highest power of any derivative" Degree = power of highest order derivative
"If \(y'\) appears inside \(\sin(y')\), find degree" If not polynomial in derivatives → degree undefined
"Equation with \(\sin y\) is linear" \(\sin y\) makes it nonlinear
"All first-order ODEs are linear" \(y' = y^2\) is nonlinear
📋 Quick Summary
  • ODE = equation with ordinary derivatives.
  • Order = highest derivative.
  • Degree = power of highest derivative (if polynomial); clear fractional powers first.
  • Linear = \(y\), derivatives first power, no products, no nonlinear functions of \(y\).
  • Superposition = if \(y_1, y_2\) solve a homogeneous linear ODE, so does \(c_1 y_1 + c_2 y_2\).
