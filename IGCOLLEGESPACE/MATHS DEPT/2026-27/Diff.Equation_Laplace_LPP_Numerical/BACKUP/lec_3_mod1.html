🎯 1. Learning Outcomes (CO1)
  • Identify and solve exact differential equations
  • Determine integrating factors for non-exact equations
  • Apply exact ODE methods to real-world problems
📌 2. Exact Differential Equations – Definition
A first-order ODE written as \[ M(x,y)\,dx + N(x,y)\,dy = 0 \] is called exact if there exists a function \(u(x,y)\) such that: \[ \frac{\partial u}{\partial x} = M, \quad \frac{\partial u}{\partial y} = N. \] The condition for exactness is: \[ \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}. \] The solution is given implicitly by \(u(x,y) = C\).
Example (Kreyszig 1.4): Solve \(2xy\,dx + (x^2 - 1)\,dy = 0\).
Here \(M = 2xy\), \(N = x^2 - 1\). Check exactness: \[ \frac{\partial M}{\partial y} = 2x, \quad \frac{\partial N}{\partial x} = 2x. \] Since they are equal, the equation is exact. Find \(u\): \[ u = \int M\,dx = \int 2xy\,dx = x^2y + h(y). \] Then, \[ \frac{\partial u}{\partial y} = x^2 + h'(y) = N = x^2 - 1. \] Thus, \[ h'(y) = -1 \quad \Rightarrow \quad h(y) = -y. \] The solution is: \[ x^2y - y = C \quad \text{or} \quad y(x^2 - 1) = C. \]
🔧 3. Integrating Factors
If \(\frac{\partial M}{\partial y} \neq \frac{\partial N}{\partial x}\), the equation is not exact. An integrating factor \(\mu(x,y)\) is a function that makes the equation exact: \[ \mu M\,dx + \mu N\,dy = 0 \quad \text{is exact.} \] Special cases:
- If \(\frac{1}{N}\left(\frac{\partial M}{\partial y} - \frac{\partial N}{\partial x}\right)\) is a function of \(x\) only, say \(f(x)\), then \[ \mu = e^{\int f(x)\,dx}. \] - If \(\frac{1}{M}\left(\frac{\partial N}{\partial x} - \frac{\partial M}{\partial y}\right)\) is a function of \(y\) only, say \(g(y)\), then \[ \mu = e^{\int g(y)\,dy}. \]
📝 Procedure for Finding Integrating Factor:
Step 1: Write the ODE as \(M dx + N dy = 0\).
Step 2: Compute \(\frac{\partial M}{\partial y}\) and \(\frac{\partial N}{\partial x}\).
Step 3: If not exact, check if \(\frac{1}{N}\left(\frac{\partial M}{\partial y} - \frac{\partial N}{\partial x}\right) = f(x)\). Then \(\mu = e^{\int f(x)dx}\).
Step 4: Multiply the equation by \(\mu\) and solve the resulting exact ODE.
📚 Workout Examples

Problems on exact ODEs and integrating factors. Click Answer to see step-by-step solution.

1. Check whether the differential equation \(-y dx + x dy = 0\) is exact. CO1
✅ Solution: Here \(M = -y\), \(N = x\).
\[ \frac{\partial M}{\partial y} = -1, \quad \frac{\partial N}{\partial x} = 1. \] Since \(-1 \neq 1\), the equation is not exact.
2. Solve the ODE: \((x^2 + y^2)dx - 2xy dy = 0\). CO1
✅ Solution: \(M = x^2 + y^2\), \(N = -2xy\).
\[ \frac{\partial M}{\partial y} = 2y, \quad \frac{\partial N}{\partial x} = -2y. \] Not exact. Compute: \[ \frac{1}{N}\left(\frac{\partial M}{\partial y} - \frac{\partial N}{\partial x}\right) = \frac{1}{-2xy}(2y - (-2y)) = \frac{4y}{-2xy} = -\frac{2}{x}. \] Integrating factor: \[ \mu = e^{\int -\frac{2}{x}dx} = e^{-2\ln x} = x^{-2}. \] Multiply: \((1 + \frac{y^2}{x^2})dx - \frac{2y}{x}dy = 0\). Now exact. Solve to get: \[ x - \frac{y^2}{x} = C \quad \text{or} \quad x^2 - y^2 = Cx. \]
3. Solve \(2xy\,dx + x^2\,dy = 0\). CO1
✅ Solution: \(M = 2xy\), \(N = x^2\).
\[ \frac{\partial M}{\partial y} = 2x, \quad \frac{\partial N}{\partial x} = 2x. \] Exact. Find \(u\): \[ u = \int M dx = x^2y + h(y). \] \[ \frac{\partial u}{\partial y} = x^2 + h'(y) = N = x^2 \Rightarrow h'(y)=0 \Rightarrow h(y)=C. \] Solution: \[ x^2y = C \quad \Rightarrow \quad y = \frac{C}{x^2}. \]
4. Solve the IVP: \((e^{x+y} + ye^y)dx + (xe^y - 1)dy = 0\), \(y(0) = -1\). CO1
✅ Solution: After checking exactness, the solution is: \[ e^{x+y} + xe^y - y = C. \] Using \(y(0) = -1\): \[ e^{-1} + 0 - (-1) = C \quad \Rightarrow \quad C = e^{-1} + 1. \] Thus, \[ e^{x+y} + xe^y - y = e^{-1} + 1. \]
5. Find an integrating factor and solve: \((x^2 + y^2 + x)dx + xy\,dy = 0\). CO1
✅ Solution: \(M = x^2 + y^2 + x\), \(N = xy\).
\[ \frac{\partial M}{\partial y} = 2y, \quad \frac{\partial N}{\partial x} = y. \] Not exact. Compute: \[ \frac{1}{N}\left(\frac{\partial M}{\partial y} - \frac{\partial N}{\partial x}\right) = \frac{1}{xy}(2y - y) = \frac{1}{x}. \] Integrating factor: \[ \mu = e^{\int \frac{1}{x}dx} = x. \] Multiply: \((x^3 + xy^2 + x^2)dx + x^2y\,dy = 0\). Solving gives: \[ \frac{x^4}{4} + \frac{x^2y^2}{2} + \frac{x^3}{3} = C. \]
6. Solve \(3(y+1)dx = 2x\,dy\). CO1
✅ Solution: Rewrite: \(3(y+1)dx - 2x\,dy = 0\). This is separable: \[ \frac{dx}{x} = \frac{2}{3(y+1)}dy. \] Integrate: \[ \ln|x| = \frac{2}{3}\ln|y+1| + C. \] \[ x = k (y+1)^{2/3}. \]
7. Solve \(y dx + (3x - y^2) dy = 0\) using integrating factor. CO1
✅ Solution: \(M = y\), \(N = 3x - y^2\).
\[ \frac{\partial M}{\partial y} = 1, \quad \frac{\partial N}{\partial x} = 3. \] Not exact. Compute: \[ \frac{1}{M}\left(\frac{\partial N}{\partial x} - \frac{\partial M}{\partial y}\right) = \frac{1}{y}(3 - 1) = \frac{2}{y}. \] Integrating factor: \[ \mu = e^{\int \frac{2}{y}dy} = e^{2\ln y} = y^2. \] Multiply: \(y^3 dx + (3xy^2 - y^4)dy = 0\). Solving gives: \[ xy^3 - \frac{y^5}{5} = C. \]
8. Solve the exact equation: \((\cos y \sinh x + 1)dx - \sin y \cosh x\,dy = 0\). CO1
✅ Solution: \(M = \cos y \sinh x + 1\), \(N = -\sin y \cosh x\).
\[ \frac{\partial M}{\partial y} = -\sin y \sinh x, \quad \frac{\partial N}{\partial x} = -\sin y \sinh x. \] Exact. Find \(u\): \[ u = \int M dx = \cos y \cosh x + x + h(y). \] \[ \frac{\partial u}{\partial y} = -\sin y \cosh x + h'(y) = N = -\sin y \cosh x \Rightarrow h'(y)=0. \] Solution: \[ \cos y \cosh x + x = C. \]
⚠️ Common Mistakes to Avoid
MistakeCorrection
Forgetting to check exactness condition \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\) Always verify before attempting exact method
Using wrong integrating factor formula Check whether \(f(x)\) or \(g(y)\) form applies
Omitting constant of integration when finding \(u\) Include \(h(y)\) or \(h(x)\) appropriately
📋 Quick Summary
  • Exact ODE: \(M dx + N dy = 0\) with \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\)
  • Solution: \(u(x,y) = \int M dx + h(y) = C\)
  • Integrating factor: \(\mu = e^{\int f(x)dx}\) or \(\mu = e^{\int g(y)dy}\)
