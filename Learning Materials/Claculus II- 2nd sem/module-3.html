📘 Detailed Teaching Notes: Reduction Formulae

📖 What is a Reduction Formula?

A reduction formula expresses an integral involving a parameter (like a power) in terms of a simpler integral of the same type with a reduced parameter. Repeated application eventually leads to a basic integral that can be evaluated directly.

💡 General Idea: If \(I_n = \int f(x,n) dx\), then a reduction formula gives \(I_n = g(x) + k \cdot I_{n-2}\) (or similar), allowing us to reduce the power step by step.

📐 1. Reduction Formula for \(\int \sin^n x \, dx\)

🔸 When \(n\) is ODD (\(n = 1, 3, 5, 7, \ldots\))

Method: Substitute \(t = \cos x\). This works because we can factor out one \(\sin x\).

Step-by-step:
1. Write \(\sin^n x = \sin^{n-1}x \cdot \sin x = (1-\cos^2 x)^{\frac{n-1}{2}} \cdot \sin x\)
2. Let \(t = \cos x\) ⇒ \(dt = -\sin x dx\) ⇒ \(\sin x dx = -dt\)
3. The integral becomes \(-\int (1-t^2)^{\frac{n-1}{2}} dt\)
4. Expand using binomial theorem and integrate term by term
5. Substitute back \(t = \cos x\)
Example: \(\int \sin^3 x dx = \int (1-\cos^2 x)\sin x dx = -\int (1-t^2)dt = -t + \frac{t^3}{3} + C = -\cos x + \frac{1}{3}\cos^3 x + C\)

🔸 When \(n\) is EVEN (\(n = 2, 4, 6, 8, \ldots\))

Reduction Formula (derived via integration by parts):

\[ \int \sin^n x \, dx = -\frac{\cos x \sin^{n-1}x}{n} + \frac{n-1}{n} \int \sin^{n-2}x \, dx \]

Derivation: Write \(\int \sin^n x dx = \int \sin x \cdot \sin^{n-1}x dx\). Let \(u = \sin^{n-1}x\), \(dv = \sin x dx\). Then \(du = (n-1)\sin^{n-2}x \cos x dx\), \(v = -\cos x\).

Example: For \(n=4\): \(\int \sin^4 x dx = -\frac{\cos x \sin^3 x}{4} + \frac{3}{4}\int \sin^2 x dx\). Then \(\int \sin^2 x dx = \frac{x}{2} - \frac{\sin 2x}{4}\).

📐 2. Reduction Formula for \(\int \cos^n x \, dx\)

🔸 When \(n\) is ODD (\(n = 1, 3, 5, 7, \ldots\))

Method: Substitute \(t = \sin x\).

Step-by-step:
1. Write \(\cos^n x = \cos^{n-1}x \cdot \cos x = (1-\sin^2 x)^{\frac{n-1}{2}} \cdot \cos x\)
2. Let \(t = \sin x\) ⇒ \(dt = \cos x dx\)
3. The integral becomes \(\int (1-t^2)^{\frac{n-1}{2}} dt\)
4. Expand and integrate
5. Substitute back \(t = \sin x\)
Example: \(\int \cos^5 x dx = \int (1-\sin^2 x)^2 \cos x dx = \int (1-2\sin^2 x+\sin^4 x)\cos x dx = \sin x - \frac{2}{3}\sin^3 x + \frac{1}{5}\sin^5 x + C\)

🔸 When \(n\) is EVEN (\(n = 2, 4, 6, 8, \ldots\))

Reduction Formula (derived via integration by parts):

\[ \int \cos^n x \, dx = \frac{\sin x \cos^{n-1}x}{n} + \frac{n-1}{n} \int \cos^{n-2}x \, dx \]

Derivation: Write \(\int \cos^n x dx = \int \cos x \cdot \cos^{n-1}x dx\). Let \(u = \cos^{n-1}x\), \(dv = \cos x dx\). Then \(du = -(n-1)\cos^{n-2}x \sin x dx\), \(v = \sin x\).

📐 3. Definite Integrals from \(0\) to \(\pi/2\)

⭐ KEY FORMULA: \[ \int_0^{\pi/2} \sin^n x \, dx = \int_0^{\pi/2} \cos^n x \, dx = \begin{cases} \frac{n-1}{n} \cdot \frac{n-3}{n-2} \cdots \frac{3}{4} \cdot \frac{1}{2} \cdot \frac{\pi}{2}, & n \text{ even}\\ \frac{n-1}{n} \cdot \frac{n-3}{n-2} \cdots \frac{4}{5} \cdot \frac{2}{3} \cdot 1, & n \text{ odd} \end{cases} \]

Proof: Using the reduction formula, the boundary term vanishes at both limits because \(\sin(0)=0\) and \(\cos(\pi/2)=0\). So \(I_n = \frac{n-1}{n} I_{n-2}\).

Base values: \(I_0 = \frac{\pi}{2}\), \(I_1 = 1\)

📐 4. Reduction Formula for \(\int \sin^p x \cos^q x \, dx\)

🔸 Case 1: \(p\) is ODD

Method: Substitute \(t = \cos x\). Write \(\sin^p x \cos^q x = \sin^{p-1}x \cos^q x \cdot \sin x\).

🔸 Case 2: \(q\) is ODD

Method: Substitute \(t = \sin x\). Write \(\sin^p x \cos^q x = \sin^p x \cos^{q-1}x \cdot \cos x\).

🔸 Case 3: \(p\) and \(q\) are BOTH EVEN

Use the reduction formula:

\[ \int \sin^p x \cos^q x dx = -\frac{\cos^{q+1}x \sin^{p-1}x}{p+q} + \frac{p-1}{p+q} \int \sin^{p-2}x \cos^q x dx \]
\[ \int \sin^p x \cos^q x dx = \frac{\sin^{p+1}x \cos^{q-1}x}{p+q} + \frac{q-1}{p+q} \int \sin^p x \cos^{q-2}x dx \]
⭐ PRODUCT FORMULA for \(\int_0^{\pi/2} \sin^p x \cos^q x dx\): \[ \int_0^{\pi/2} \sin^p x \cos^q x dx = \frac{(p-1)(p-3)\cdots (q-1)(q-3)\cdots}{(p+q)(p+q-2)\cdots} \times K \] where \(K = \frac{\pi}{2}\) if both \(p\) and \(q\) are even, otherwise \(K = 1\).
Example 1: \(\int_0^{\pi/2} \sin^5 x \cos^6 x dx = \frac{4\cdot2\cdot5\cdot3\cdot1}{11\cdot9\cdot7\cdot5\cdot3\cdot1} = \frac{8}{693}\)
Example 2: \(\int_0^{\pi/2} \sin^6 x \cos^8 x dx = \frac{5\cdot3\cdot1\cdot7\cdot5\cdot3\cdot1}{14\cdot12\cdot10\cdot8\cdot6\cdot4\cdot2} \cdot \frac{\pi}{2} = \frac{5\pi}{4096}\)

📐 5. Reduction Formula for \(\int \tan^n x \, dx\) and \(\int \cot^n x \, dx\)

🔸 For \(\int \tan^n x dx\) (n ≥ 2)

Derivation: \(\int \tan^n x dx = \int \tan^{n-2}x \tan^2 x dx = \int \tan^{n-2}x (\sec^2 x - 1) dx\)
\[ \int \tan^n x dx = \frac{\tan^{n-1}x}{n-1} - \int \tan^{n-2}x dx \]
Base case: \(\int \tan x dx = \ln|\sec x| + C\)

🔸 For \(\int \cot^n x dx\) (n ≥ 2)

Derivation: \(\int \cot^n x dx = \int \cot^{n-2}x \cot^2 x dx = \int \cot^{n-2}x (\csc^2 x - 1) dx\)
\[ \int \cot^n x dx = -\frac{\cot^{n-1}x}{n-1} - \int \cot^{n-2}x dx \]
Base case: \(\int \cot x dx = \ln|\sin x| + C\)

🔸 Definite Integrals with \(\tan^n x\)

For \(\phi(n) = \int_0^{\pi/4} \tan^n x dx\): \[ \phi(n) + \phi(n-2) = \frac{1}{n-1} \]
For \(I_n = \int_0^{\pi/3} \tan^n x dx\): \[ (n-1)(I_n + I_{n-2}) = (\sqrt{3})^{n-1} \]

📐 6. Other Important Reduction Formulae

🔸 For \(\int x^m \sin(nx) dx\)

\[ \int x^m \sin(nx) dx = -\frac{x^m \cos(nx)}{n} + \frac{m x^{m-1} \sin(nx)}{n^2} - \frac{m(m-1)}{n^2} \int x^{m-2} \sin(nx) dx \]

🔸 For \(\int x^m \cos(nx) dx\)

\[ \int x^m \cos(nx) dx = \frac{x^m \sin(nx)}{n} + \frac{m x^{m-1} \cos(nx)}{n^2} - \frac{m(m-1)}{n^2} \int x^{m-2} \cos(nx) dx \]

🔸 For \(\int x^m (\log x)^n dx\)

\[ \int x^m (\log x)^n dx = \frac{x^{m+1}}{m+1} (\log x)^n - \frac{n}{m+1} \int x^m (\log x)^{n-1} dx \]

🔸 For \(\int \frac{x^n}{(\log x)^m} dx\)

\[ \int \frac{x^n}{(\log x)^m} dx = -\frac{x^{n+1}}{(m-1)(\log x)^{m-1}} + \frac{n+1}{m-1} \int \frac{x^n}{(\log x)^{m-1}} dx \]

🔸 Gamma Function Result

\[ \int_0^{\infty} x^n e^{-x} dx = n! \quad \text{(for positive integer n)} \]

Solution 1: \(\int x^2 e^{ax} dx\)
Step 1: Integration by parts: \(\int u dv = uv - \int v du\).
Step 2: Let \(u = x^2\), \(dv = e^{ax}dx\) → \(du = 2x dx\), \(v = e^{ax}/a\).
Step 3: \(\int x^2 e^{ax}dx = \frac{x^2 e^{ax}}{a} - \frac{2}{a}\int x e^{ax}dx\).
Step 4: For \(\int x e^{ax}dx\): let \(u = x\), \(dv = e^{ax}dx\) → \(du = dx\), \(v = e^{ax}/a\).
Step 5: \(\int x e^{ax}dx = \frac{x e^{ax}}{a} - \frac{1}{a}\int e^{ax}dx = \frac{x e^{ax}}{a} - \frac{e^{ax}}{a^2}\).
Step 6: Substitute back: \(\frac{x^2 e^{ax}}{a} - \frac{2}{a}(\frac{x e^{ax}}{a} - \frac{e^{ax}}{a^2}) = \frac{e^{ax}}{a}(x^2 - \frac{2x}{a} + \frac{2}{a^2}) + C\).
\(\boxed{\frac{e^{ax}}{a}\left(x^2 - \frac{2x}{a} + \frac{2}{a^2}\right) + C}\)

Step 1: \(\sin^2 mx = \frac{1-\cos 2mx}{2}\). So \(\int_0^{\pi} \sin^2 mx dx = \frac{1}{2}\int_0^{\pi} dx - \frac{1}{2}\int_0^{\pi} \cos 2mx dx = \frac{\pi}{2} - 0 = \frac{\pi}{2}\).
Step 2: For \(m \neq n\): \(\sin mx \sin nx = \frac{1}{2}[\cos(m-n)x - \cos(m+n)x]\).
Step 3: \(\int_0^{\pi} \cos(kx) dx = 0\) for integer \(k \neq 0\).
Step 4: Therefore \(\int_0^{\pi} \sin mx \sin nx dx = 0\) for \(m \neq n\). Proved.
\(\boxed{\int_0^{\pi} \sin^2 mx dx = \frac{\pi}{2}, \quad \int_0^{\pi} \sin mx \sin nx dx = 0 \text{ for } m \neq n}\)
